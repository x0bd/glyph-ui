GlyphUI Hooks Demo

This page demonstrates the React-like hooks system implemented in GlyphUI.

Implementation Notes:

This demo showcases several hooks:

  • useState - For state management in functional components
  • useEffect - For side effects (timers, DOM updates, etc.)
  • useRef - For persistent values across renders
  • useMemo - For memoizing expensive calculations
  • useCallback - For memoizing callbacks
